# 《我们都一样》本地可视化原型 v1

这是一个本地可运行的反事实人生生成器原型。

## 怎么运行

直接双击 `index.html`，用浏览器打开即可。

不需要安装依赖，不需要联网。

## 当前已经实现

1. MBTI 四维输入
   - 每一维可以直接选字母
   - 有默认强度：稍微偏向、明显偏向、非常偏向
   - 可以自定义 50-100 的偏向数值

2. 当前表现校准题
   - 8 道题
   - 使用 7 级程度选择
   - 用来修正 MBTI 无法覆盖的部分

3. 成长环境题库
   - 24 道题
   - 8 个成长环境维度
   - 使用 7 级程度选择

4. 平行成长剧本
   - 首富的孩子
   - 顶流巨星家的小孩
   - 神秘老钱家族继承人
   - 海边小镇野生长大的小孩
   - 全家都是卷王的天选打工圣体
   - 被亲戚轮流带大的江湖型小孩

5. 结果页
   - 现在呈现出来的你
   - 成长环境训练出的部分
   - 剧本外的你
   - 平行成长剧本结果
   - 关键成长事件
   - 离谱人生小传
   - 温情收尾

6. 后台调试面板
   - MBTI 初始人格表现向量
   - 校准后的当前人格表现向量
   - 现实成长环境向量
   - 现实环境塑形影响向量
   - 反推出厂底色向量
   - 新剧本塑形影响向量
   - 平行人生人格变化向量

## 当前模型说明

本原型不是心理测评，也不输出真实人格结论。

它的核心逻辑是：

```text
当前人格表现 = 出厂底色 + 现实环境塑形
出厂底色 = 当前人格表现 - 部分现实环境塑形
平行人生人格 = 出厂底色 + 新剧本塑形
```

目前现实环境只剥离 35%，新剧本塑形按 55% 加权。这是为了避免过度“还原真我”，同时保留娱乐化推演效果。

## 后续重点

1. 检查 MBTI 到 8 维后台人格向量的映射是否合理
2. 检查成长环境题是否过于私人或冒犯
3. 检查不同剧本是否真的生成了不同结果
4. 检查同一剧本对不同用户是否有差异
5. 调整环境塑形矩阵
6. 增加结果页截图功能
7. 增加匿名反馈收集
