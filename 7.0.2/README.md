# BBTI 7.0.2｜小动物结果版

本版本基于 7.0.1.3 的稳定模型，不改题目、不改 33 题权重、不改匹配算法。

## 本次更新

- 将 P01–P12 的前台名称替换为最终小动物昵称。
- 新增 `animalProfiles` 数据：动物、动物名称、卡片名、slogan、核心解释、成长解释。
- 结果页新增可截图的小动物身份卡。
- 结果页正文改为两段解释：核心解释 + 成长解释。
- 副标签、距离排名、维度向量、反向修正等模型信号保留在默认折叠的调试区。

## 网页必须文件

- `index.html`
- `styles.css`
- `app.js`
- `model-data.js`
- `model-core.js`

## 检测脚本

运行：

```bash
node scripts/check-model-distribution.js
```

该脚本用于验证模型分布，和前台小动物文案无关。
