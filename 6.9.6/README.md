# 返老还童 / 我们都一样 v3.4

这是一个纯前端静态网站版本，可直接上传 GitHub 仓库并用 GitHub Pages、Vercel、Netlify 等部署。

## 文件

- `index.html`
- `styles.css`
- `app.js`

无需构建工具，无需后端，直接打开 `index.html` 即可本地试玩。

## 本版重点

- 结果页从“性格测试结果”改为“初始化设定反推报告”。
- 主结构：现实输入 → 删除清单 → 保留清单 → 初始化设定。
- 物件库改为荒诞生活用品方向。
- 增加语气切换：`温柔地看` / `劳资爱怎么看怎么看`。
- 仅本地导出匿名 JSON，不主动上传数据。
- 分享图不放人生剧本，先聚焦初始化设定。

## 隐私提醒

当前代码不会主动收集姓名、手机号、定位、IP 或开放题原文。导出 JSON 仅保存在用户本地。
如果部署到第三方平台，平台本身可能有访问日志；公开测试前建议在页面补充更正式的隐私说明。

## 部署建议

把本文件夹全部上传到仓库根目录，设置 GitHub Pages 的发布源为仓库根目录或对应分支即可。
