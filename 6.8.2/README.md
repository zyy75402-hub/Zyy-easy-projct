# 《我们都一样》本地可视化原型 v3

## 运行方式

解压后直接双击 `index.html`，用浏览器打开即可运行。当前版本为纯前端本地原型，不需要安装依赖，不上传数据。

## v3 主要变化

1. 删除独立环境评分系统，统一改为两套规则：
   - `mbtiRule`：用于反推初始化 MBTI。
   - `objectRule`：用于匹配物件人格标签。
   - `evidenceTags`：只用于最后解释“哪些场景反馈影响最大”，不单独形成第三套评分。
2. 后续题目全部改成七级程度题，不再出现多选题式问法。
3. 反推题从“我现在怎样 / 我意识到什么”改为“过去发生过什么场景，别人通常如何反馈”。
4. 增加物件标签系统，输出格式改为：`初始化 MBTI - 物件标签`。
5. 增加每题答题用时记录，用于后续判断题目是否难懂或用户是否纠结。答题时间不直接参与人格计算。
6. 增加本地匿名 JSON 导出，方便小范围测试后分析，不包含姓名、联系方式、位置或自述原文。
7. 全部“不确定”时不会强行制造变化，会显示较低反推置信度。

## 当前版本重点验证

- 用户是否理解“现实 MBTI”和“初始化 MBTI”的区别。
- 反推题是否真的像在回忆过去场景，而不是重新测试 MBTI。
- 物件标签是否比传统人格底色更亲切、有趣。
- 同一 MBTI 在不同回答下是否能得到不同物件。
- 同一物件进入不同剧本后，平行人生小传是否有区别。

## 隐私说明

本地版默认不上传任何数据。点击“导出本地匿名数据”时，只会生成本地 JSON 文件，内容包括：MBTI 输入、匿名答案、答题用时、结果和证据标签。不包含姓名、联系方式、精确位置、学校、照片或自述原文。
