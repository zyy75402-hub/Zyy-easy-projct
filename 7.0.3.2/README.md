# BBTI 7.0.3.2｜前台文案与结果页标签优化

本版本基于 `7.0.3.1` 小版本优化。保留 33 题人格测试、P01-P12 匹配算法、平行人生环境变量映射、事件选择算法和验证脚本核心逻辑，只优化前台展示文案、经历阶段表达和小动物结果页部分标签。

## 本次更新

- 小动物主卡片不再显示画面名 / 图片设定文案，只保留动物图像、昵称、动物类别和 slogan。
- 小动物结果页「当前输入」改为「测试前的你」。
- 小动物结果页「系统校准后的当前外壳」改为「你现在表现出来的一面」。
- 平行人生经历的时间表达改为更具体的年龄 / 阶段描述，志愿、城市、大学、离家类事件不再误写成「青春期那几年」。
- 经历桥接句增加变化，减少同一组结果中重复模板感。

## 保持不变

- 不改 33 题人格测试题目。
- 不改 33 题权重。
- 不改 P01-P12 原始人格匹配算法。
- 不改 7.0.2 小动物昵称、slogan、结果解释文案。
- 不改 7.0.3 环境变量映射表。
- 不改 7.0.3 事件主题匹配逻辑。
- 不改平行人生事件选择算法。
- 不改验证脚本的核心验收逻辑。
- 不接 AI API。
- 不接数据库。
- 不做保存卡片功能。

## 本地运行

直接用静态服务器打开本目录即可，例如：

```bash
python3 -m http.server 7104
```

然后访问：

```text
http://127.0.0.1:7104/7.0.3.2/
```

```bash
node scripts/check-model-distribution.js
node scripts/check-parallel-life-env-mapping.js
node scripts/check-parallel-life-event-selection.js
```
